Personal Projects

Cashister
Cashister is an iPad cash register application I created after recognizing the challenges of working in a high school concession stand. The application allows workers to more efficiently fulfill larger orders by providing a list of items ordered and calculating the order's total cost. The application is currently available on the iPad App Store.
Extreme Tic Tac Toe
For the CornHacks 2019 hackathon, my team created the game of Extreme Tic Tac Toe (also known as Ultimate Tic Tac Toe) using Java. The hackathon was a great learning experience of how to effectively work with multiple people in one codebase as well as how to effectively utilize pair programming.
Class-Learn
Class-Learn is an open source learning platform I have slowly been working on. The focus of the project is to create a learning platform optimized for both the student and teacher through providing customization of how courses are set up. So far, however, I have only worked on setting up authentication and permissions for users.
